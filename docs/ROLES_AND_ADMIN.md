# 2 Roles: Main Admin & Limited User (Admin Approval)

## Overview

- **Main admin** – Full access. Manages entire website from Admin Panel. Only admin can **approve** new users and give them access.
- **New users (limited)** – After **admin approval**, they get access to: Live Auctions (bid), My Car Tracking (live data), Payments, Browse Cars, Profile, Settings. Until approved, they only see “Account Pending Approval”.

---

## 1. Main Admin

- **Who:** First person who registers becomes the main admin (only one).
- **Access:** Admin Panel (`/admin`):
  - **Users** – Approve/reject new users (only admin can give access).
  - **Vehicles** – Add/edit cars.
  - **Auctions** – Create/manage live and upcoming auctions.
  - **Site Settings, Page Content, FAQ, Testimonials, SEO, Blog, Pricing, Inquiries** – Manage full website from here.
- **Dashboard home:** Shows “User approval” card with pending count and link to **Manage & approve users**.

---

## 2. New Users (Limited)

- **Register:** Anyone can register (Sign up).
- **Right after signup:** Status = **Pending**. They are redirected to **“Account Pending Approval”** and cannot use Dashboard menu (no Live Auctions, Tracking, Payments) until approved.
- **After admin approves:** They get access to:
  - **Dashboard** – Overview (saved vehicles, inquiries, imported vehicles, total spent).
  - **Live Auctions** – See and **place bids**.
  - **My Car Tracking** – See **where their car has reached (live data)**.
  - **Payments** – View and pay for their orders.
  - **Browse Cars, Profile, Settings** – Limited to their own data.
- They **cannot** open Admin Panel or manage other users/site content.

---

## 3. How to Approve a User (Admin)

1. Log in as **admin**.
2. Go to **Admin Panel** → **Users** (or use “Manage & approve users” on admin dashboard).
3. In **“Pending approval”** you will see new signups.
4. Click **Approve** for a user.
5. That user can refresh the site; they will then see the full limited menu (Live Auctions, Car Tracking, Payments, etc.).

---

## 4. Summary

| Role        | Who                    | Access                                                                 |
|------------|------------------------|-------------------------------------------------------------------------|
| **Main admin** | First registered user  | Full Admin Panel: users, vehicles, auctions, site content, settings.   |
| **User (pending)** | New signups            | Only “Account Pending Approval” page until admin approves.             |
| **User (approved)** | After admin approval   | Dashboard, Live Auctions (bid), Car Tracking (live), Payments, etc.   |
