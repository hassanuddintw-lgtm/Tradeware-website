# Tradeware UI - Japanese Car Import Marketplace

A high-end, production-ready UI/UX MVP for a Japanese car import marketplace.

## Features

- Complete multi-page website structure
- Fully responsive design
- SEO-optimized pages
- Admin panel UI (frontend only)
- Realistic automotive data and content
- Modern, enterprise-grade design

## Technology Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
├── app/                    # Next.js App Router pages
│   ├── (public)/          # Public pages
│   ├── admin/             # Admin UI pages
│   └── layout.tsx         # Root layout
├── components/            # Reusable UI components
│   ├── ui/               # Base UI components
│   ├── layout/           # Layout components
│   └── features/         # Feature-specific components
├── lib/                   # Utilities and helpers
├── types/                 # TypeScript type definitions
└── data/                  # Mock data and content
```
